# Teto Territory 
#### (AKA Rainbet 2 AKA cool-game AKA Weegee is drunk again)

## CONTROLS
| Key | Action |
| -- | -- |
| WASD | Movement |
| Space | Brake |
| F | Rocket |
| G | Gun |
| K | Toggle Weegee Mode |
| T | Toggle foreground visiblity |
| Q | Toggle bg/fg drawing | 
| H | Spawn Horse Race Test (WIP) |
| 1-9 | Spawn Item |

## BUILDING
1. Install SDL3, SDL3_img, and tinyxml2 on your system.
2. run `make`
